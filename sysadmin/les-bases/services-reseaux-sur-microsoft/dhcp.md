# DHCP

Le Protocole de configuration dynamique d'hôte (Dynamic Host Configuration Protocol, DHCP) est un protocole réseau utilisé pour attribuer automatiquement des adresses IP, des masques de sous-réseau, des passerelles par défaut et d'autres paramètres réseau à des ordinateurs et autres appareils connectés à un réseau.

Exemple d'utilisation de DHCP sur un serveur Windows :

1. Installation du service DHCP : Il est possible d'installer le service DHCP sur un serveur Windows en utilisant le Gestionnaire de serveur ou en utilisant la ligne de commande.
2. Configuration des plages d'adresses IP : Une fois le service DHCP installé, il est nécessaire de configurer les plages d'adresses IP qui seront attribuées aux ordinateurs et autres appareils connectés au réseau.
3. Configuration des options DHCP : Il est également possible de configurer les options DHCP telles que les paramètres de passerelle par défaut, les paramètres de masque de sous-réseau et d'autres paramètres réseau.
4. Test de la configuration : Une fois la configuration terminée, il est possible de tester la configuration en connectant un ordinateur au réseau et en vérifiant que l'ordinateur reçoit une adresse IP automatiquement via DHCP.

Dans une situation d'application, le service DHCP peut être utilisé pour gérer automatiquement les adresses IP sur un réseau en permettant aux ordinateurs et autres appareils de recevoir automatiquement une adresse IP valide sans intervention humaine. Cela simplifie la gestion du réseau en évitant les erreurs d'adresse IP et en facilitant la configuration des ordinateurs et des appareils connectés au réseau.
